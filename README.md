# Siete días para ti

Demo gratuita de una experiencia de siete vídeos desbloqueables mediante
preguntas tipo test.

## Probar en local

Abre `index.html` directamente o inicia cualquier servidor web estático en
esta carpeta.

La aplicación no necesita instalar dependencias. El progreso se guarda en el
navegador mediante `localStorage`.

## Personalizar

Las preguntas, respuestas, vídeos y colores están definidos al principio de
`app.js`. Cada vídeo usa su identificador público de YouTube.
